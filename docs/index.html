Bullet_Hell_Game

Arcade style space shooter Android Game made using Unity and C#.

Download this project as a .zip file

Features of Bullet_Hell_Game

  1. Uses Google Firebase for authentication services.

  2. User can create an account, which is displayed in the leaderboards.

  3. Scrapes Authentic sounds made using Openmpt.

  4. Background created using Blender.

  5. You use your phone's accelorometer to change direction of ship and tap anywhere on the screen to shoot. Your aim is to kill as many enemies as possible and beat the high score.
